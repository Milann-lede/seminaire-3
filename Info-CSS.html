Qu'est-ce que le CSS ?

Le langage de feuilles de style en cascade

Le CSS (Cascading Style Sheets) est un langage de feuilles de style utilisé pour décrire la présentation d'un document écrit en HTML. Il permet de contrôler la mise en page, les couleurs, les polices et bien d'autres aspects visuels d'une page web.


Structure de base d'une règle CSS


/* Sélecteur */
element {
    propriété: valeur; /* Déclaration */
}

/* Exemple */
h1 {
    color: blue;
    font-size: 2rem;
    margin-bottom: 1rem;
}

.container {
    width: 80%;
    margin: 0 auto;
    padding: 20px;
    background-color: #f0f0f0;
}

Sélecteurs de base

Sélecteurs d'éléments (p, h1, div), de classe (.ma-classe) et d'identifiant (#mon-id).

Sélecteurs combinatoires

Descendant (espace), enfant direct (>), adjacent (+), et frère général (~).

Sélecteurs d'attributs

[attribut], [attribut=valeur], [attribut^=valeur], [attribut$=valeur], etc.

Pseudo-classes et éléments

:hover, :focus, :nth-child(), ::before, ::after pour des effets avancés.

Couleur

color, background-color, font-family, font-size, font-weight, text-align, line-height.

CSS3 et fonctionnalités avancées

Des fonctionnalités modernes pour des designs impressionnants

CSS3 a introduit de nombreuses fonctionnalités avancées qui permettent de créer des designs plus complexes et attrayants sans avoir recours à des images ou à JavaScript.

Fonctionnalités CSS3 principales

  • Animations et transitions : transition, animation, @keyframes
  • Transformations : transform (rotate, scale, translate, skew)
  • Dégradés : linear-gradient(), radial-gradient()
  • Ombres : box-shadow, text-shadow
  • Flexbox : Un modèle de mise en page unidimensionnel
  • Grid : Un modèle de mise en page bidimensionnel
  • Variables CSS : --nom-variable pour une maintenance facilitée
  • Media Queries : Pour le responsive design

Bonnes pratiques CSS

Conseils pour un CSS de qualité

  • Utilisez une méthodologie cohérente (BEM, OOCSS, etc.)
  • Organisez vos règles CSS de manière logique
  • Utilisez des variables CSS pour les valeurs récurrentes
  • Évitez les sélecteurs trop spécifiques
  • Optimisez pour les performances (évitez les règles coûteuses)
  • Pensez mobile-first pour le responsive design
  • Utilisez des préprocesseurs (Sass, Less) pour les projets complexes
  • Validez votre code avec des outils comme CSS Lint

Responsive Design

Le responsive design consiste à créer des sites qui s'adaptent à toutes les tailles d'écran. Utilisez les media queries, les unités relatives (%, rem, vw/vh) et des layouts flexibles (Flexbox, Grid).
