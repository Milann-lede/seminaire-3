Qu'est-ce que le HTML ?

Le langage de balisage hypertexte

Le HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour créer des pages web. Il décrit la structure d'une page web et permet d'intégrer du contenu comme du texte, des images, des vidéos et des liens hypertextes.


Structure de base d'un document HTML


 <!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Titre de la page</title>
</head>
<body>
    <h1>Mon titre principal</h1>
    <p>Un paragraphe de texte.</p>
</body>
</html>

Les éléments HTML essentiels

Les balises fondamentales pour structurer votre contenu

Balises de titre

Les balises <h1> à <h6> définissent les titres et sous-titres, avec <h1> étant le plus important.

Paragraphes et texte

La balise <p> pour les paragraphes, <strong> pour le texte important, <em> pour le texte en emphase.

Liens et images

<a> pour les liens hypertextes, <img> pour intégrer des images avec l'attribut src.

Listes

<ul> pour les listes non ordonnées, <ol> pour les listes ordonnées, <li> pour les éléments de liste.

HTML5 Sémantique

Donnez du sens à votre structure

HTML5 a introduit des balises sémantiques qui décrivent plus précisément le contenu qu'elles entourent, améliorant l'accessibilité et le référencement.

Balises sémantiques principales

  • <header> : En-tête de page ou de section
  • <nav> : Section de navigation
  • <main> : Contenu principal de la page
  • <article> : Contenu autonome et indépendant
  • <section> : Section thématique de contenu
  • <aside> : Contenu complémentaire
  • <footer> : Pied de page ou de section

L'utilisation de ces balises aide les moteurs de recherche et les technologies d'assistance à mieux comprendre la structure de votre page.
