Qu'est-ce que JavaScript ?

Le langage de programmation pour le web interactif

JavaScript est un langage de programmation de haut niveau, interprété et orienté objet qui permet de créer du contenu web dynamique et interactif. Il est l'un des trois piliers du développement web avec HTML et CSS.


Structure de base d'un script JavaScript


// Commentaire sur une ligne
/* Commentaire 
   multi-lignes */

// Déclaration de variable
let message = "Bonjour le monde!";
const PI = 3.14159;

// Fonction simple
function afficherMessage() {
    console.log(message);
    alert(message);
}

// Appel de fonction
afficherMessage();

// Structure conditionnelle
if (PI > 3) {
    console.log("PI est plus grand que 3");
} else {
    console.log("PI est plus petit que 3");
}

// Boucle
for (let i = 0; i < 5; i++) {
    console.log("Itération: " + i);
}

Manipulation du DOM

Interagir avec les éléments de la page web

Le DOM (Document Object Model) est une interface de programmation qui représente la structure d'un document HTML/XML et permet à JavaScript de manipuler le contenu, la structure et le style d'une page web.

Méthodes DOM essentielles

  • document.getElementById() : Sélectionne un élément par son ID
  • document.querySelector() : Sélectionne le premier élément correspondant au sélecteur CSS
  • document.querySelectorAll() : Sélectionne tous les éléments correspondant au sélecteur CSS
  • element.innerHTML : Lit ou modifie le contenu HTML d'un élément
  • element.textContent : Lit ou modifie le contenu texte d'un élément
  • element.style : Modifie les styles CSS d'un élément
  • element.addEventListener() : Attache un gestionnaire d'événements

APIs Web JavaScript

Interagir avec le navigateur et les fonctionnalités du web

APIs couramment utilisées

  • Fetch API : Effectuer des requêtes HTTP
  • LocalStorage/SessionStorage : Stockage de données côté client
  • Geolocation API : Accéder à la localisation de l'utilisateur
  • Canvas API : Dessiner et animer des graphiques
  • Web Audio API : Traitement et contrôle audio
  • Intersection Observer API : Détecter la visibilité des éléments
  • Web Workers : Exécuter du code en arrière-plan

Exemple avec Fetch API

// Récupérer des données depuis une API
fetch('https://api.exemple.com/data')
    .then(response => {
        if (!response.ok) {
            throw new Error('Erreur réseau');
        }
        return response.json();
    })
    .then(data => {
        console.log('Données reçues:', data);
        // Traiter les données
    })
    .catch(error => {
        console.error('Erreur:', error);
    });

// Version avec async/await
async function fetchData() {
    try {
        const response = await fetch('https://api.exemple.com/data');
        const data = await response.json();
        console.log('Données reçues:', data);
    } catch (error) {
        console.error('Erreur:', error);
    }
}

Bonnes pratiques JavaScript

Codez efficacement et professionnellement

Conseils pour un JavaScript de qualité

  • Utilisez const et let au lieu de var
  • Écrivez des fonctions pures quand c'est possible
  • Utilisez des noms de variables et fonctions significatifs
  • Évitez les fonctions trop longues (max 20-30 lignes)
  • Gérez proprement les erreurs avec try/catch
  • Utilisez les promesses ou async/await pour le code asynchrone
  • Commentez votre code de manière appropriée
  • Utilisez un linter (ESLint) et un formateur (Prettier)

Performance et optimisation

Pour des applications JavaScript performantes : évitez les boucles imbriquées coûteuses, utilisez le debouncing pour les événements fréquents, minimisez les manipulations DOM, et utilisez les Web Workers pour les tâches intensives.
